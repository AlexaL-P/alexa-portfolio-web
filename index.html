About

Hi! My name is Alexa Luck-Power (yep, family names) and I'm a web development student. This portfolio page is the first deployed app I have written from the ground up in HTML and CSS!

My tech related hobbies include tinkering with game consoles and arcade machines, either for servicing the hardware or to run homebrew software. I've also delivered several workshops on the BBC BASIC language - both on original 80s computers and the Pico implementation of Basic 5 for Raspberry Pi! Although my favourite has to be putting on large exhibions of vintage hardware for the public to enjoy :)
